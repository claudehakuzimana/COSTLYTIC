🚀 QUICK START

  1. Start MongoDB:
     docker run -d -p 27017:27017 mongo:7-jammy

  2. Start Redis:
     docker run -d -p 6379:6379 redis:7-alpine

  3. Seed Database:
     cd server && npm run seed

  4. Start Backend:
     npm run dev

  5. Start Frontend (new terminal):
     cd client && npm run dev

  6. Open Browser:
     http://localhost:3000

  7. Login:
     Email: admin@acme.com
